# 📊 Financial Analytics Dashboard (Power BI)

## 🚀 Project Overview

This project is a comprehensive Financial Analytics Dashboard built using Power BI. It provides insights into sales, profitability, and future forecasting to support data-driven decision-making.

---

## 📌 Key Features

* 📈 Sales & Revenue Analysis
* 💰 Profitability Insights
* 🌍 Regional Performance
* 📊 KPI Tracking (Sales, Growth %, Profit Margin)
* 🔮 Sales Forecasting with Confidence Interval
* 🧠 Dynamic Insights using DAX

---

## 📊 Dashboard Pages

1. Executive Summary
2. Profitability Analysis
3. Sales Trend Analysis
4. Forecasting Visualization

---

## 🛠️ Tools & Technologies

* Power BI
* Excel / CSV
* DAX (Data Analysis Expressions)
